Un párrafo es un bloque de texto que se utiliza para organizar y presentar información de manera clara y estructurada. En HTML, los párrafos se crean utilizando la etiqueta "< p >". Esta etiqueta indica el inicio y el final de un párrafo, permitiendo que el contenido dentro de ella se muestre como un bloque separado del resto del texto.
